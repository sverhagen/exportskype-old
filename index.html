Export-skype

Little tool to export Skype chat history to Trillian XML log format

Download executable

Export Skype

Little command-line tool to export Skype chat history to another format. Implemented format is Trillian XML log format, but other formats may easily be supported by adding implementations of the OutputHandler interface.

License

Licensed under the Apache License, thus making it free software and distributed "as-is".

Usage

Make sure you have Java installed. Download an executable JAR file and execute it; either:

  • By double-clicking on the executable JAR file
  • In a Command Prompt. This method allows you to see the progress
    java -jar <path to executable JAR file>/exportskype-<version>.jar

Limitations

  • Only tested for Windows
  • No graphical user interface (command line output only)
  • If more than one Skype account found, only the first one is used
  • Chats with multiple partners are only exported for one of those partners; i.e. very limited multi-partner support
  • Based on Skype internal database, and not on its API, which may thus be vulnerable for regression

Other Formats

Other formats than Trillian XML log format may easily be supported by adding implementations of the OutputHandler interface
